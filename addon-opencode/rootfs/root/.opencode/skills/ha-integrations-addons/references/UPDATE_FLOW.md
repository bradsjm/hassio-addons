# Update Flow

## Recommended steps

1) Check updates with `ha_get_updates`.
2) Review categories (core, add-ons, devices, HACS).
3) Apply updates in order of impact.

## Notes

- Core updates may require restart.
- Add-on updates require HA OS/Supervised.

Add-on store navigation example from the Add-ons docs:
